# PRODUCT_SPEC

## Vision
Lume Codex Cloud is a web platform for cloud coding tasks with parallel/background execution, reviewable evidence, and connector-driven delegation (GitHub, Slack, Linear).

## Personas
- Staff engineer running code changes in parallel across repos.
- Tech lead reviewing diffs, tests, and PR lifecycle.
- Engineering manager tracking usage/credits and policy compliance.

## Product Goals
- Launch Ask/Code tasks from one composer.
- Keep task execution alive in background with realtime progress.
- Review summary/logs/diff/tests/artifacts before PR.
- Continue tasks with follow-up preserving context.
- Govern environments, internet policy, and review policies per repo/workspace.

## Scope
- Auth, onboarding, cloud task dashboard, task details, environments, connectors, code review, usage/credits, analytics/admin, shortcuts.

## UX Principles
- Dark-first, keyboard-first, high signal/noise ratio.
- Evidence-first workflow: summary -> logs -> diff -> tests -> PR.
- Explicit risk labels for internet policy and setup-only secrets.
